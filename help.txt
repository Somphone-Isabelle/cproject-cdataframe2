# GIT

## processus
1. git pull
2. (modifier vos fichiers)
3. git status (facultatif, permet d'affichier le status)
4a. git add . (pour ajouter tous les fichiers)
4b. git add nom_du_fichier (pour ajouter un ou plusieurs fichiers)
5. git commit -m "Commentaire"
6. git push
